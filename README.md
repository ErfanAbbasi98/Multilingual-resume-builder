# 📄 Multilingual Resume Builder

> A free, browser-based resume builder that supports multiple languages — no sign-up, no backend, just your resume.

[![Live Demo](https://img.shields.io/badge/Live%20Demo-GitHub%20Pages-blue?style=for-the-badge)](https://erfanabbasi98.github.io/Multilingual-resume-builder)
[![License](https://img.shields.io/badge/License-MIT-green?style=for-the-badge)](LICENSE)

---

## ✨ Features

- 🌍 **Multilingual Support** — Build your resume in Persian, English, German, and more
- 📄 **PDF Export** — Download a print-ready version of your resume instantly
- ⚡ **No Installation** — Runs entirely in the browser, zero dependencies
- 🔒 **Privacy First** — All data stays on your device, nothing is sent to a server
- 📱 **Responsive** — Works on desktop and mobile
- 🎨 **Clean Design** — Professional layout ready for real-world job applications

## 📦 Getting Started

### Use Online
Simply visit the [live demo](https://YOUR_USERNAME.github.io/resume-builder/) — no setup needed.

### Run Locally

```bash
git clone https://github.com/ErfanAbbasi98/Multilingual-resume-builder.git
cd Multilingual-resume-builder
# Open index.html in your browser
open index.html
```
